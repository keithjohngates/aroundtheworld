This page will explain how to build and host a website

The following is a list of the different services, programs and applications that we will use:

Scaleway - Cloud computing service that will host the webserver and website

Debian - Linux operating system that we will use on our cloud instance

Python - Programming language to build the website

Pycharm - Programming development environment

PuTTY - SSH client

Git - Version control system

Flask - Specifc Python framework used to build the website

Nginx - Reverse proxy server

Gunicorn - Webserver

Supervisor - Process control system
